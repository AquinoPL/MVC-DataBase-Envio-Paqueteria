# MVC - Web de Gestión para Empresa de Paquetería

## Descripción
Desarrollo de una aplicación web para la gestión de envíos y clientes de una empresa de paquetería similar al modelo de negocio de Shalom. El sistema permite registrar envíos, almacenar información de clientes y realizar el seguimiento básico de paquetes, utilizando una base de datos relacional para el control y organización de la información.

## Tecnologías usadas
- Lenguajes: PHP, HTML, CSS
- Base de datos: MySQL
- Herramientas: XAMPP, Visual Studio Code
- Control de versiones: Git, GitHub

## Funcionalidades
- Registro de clientes
- Registro de envíos de paquetería
- Asignación de código de seguimiento
- Consulta de estado del envío
- Almacenamiento de información en base de datos MySQL
- Validación de formularios para el ingreso de datos

## Cómo ejecutar el proyecto
1. Instalar XAMPP o un servidor local compatible con PHP y MySQL.
2. Clonar el repositorio desde GitHub.
3. Importar la base de datos MySQL mediante phpMyAdmin.
4. Configurar la conexión a la base de datos en los archivos PHP.
5. Iniciar los servicios Apache y MySQL.
6. Acceder al sistema desde el navegador web.
